# Swiggy Sales Analysis using SQL

This project performs an end-to-end SQL-based analysis on Swiggy sales data.
It covers data validation, data modeling, KPI generation, and business insights.

## Project Contents
- `dataset/` : Raw Swiggy sales dataset (CSV)
- `sql_queries/` : PDF containing all SQL queries with explanations

## Key SQL Work Done
- Data validation and cleaning (nulls, blanks, duplicates)
- Removal of duplicate records using CTE and ROW_NUMBER()
- Star schema design with fact and dimension tables
- Data insertion into dimension and fact tables
- KPI calculation (orders, revenue, ratings)
- Deep-dive business analysis and trend analysis

## SQL Concepts Used
- CTEs
- Window Functions
- Aggregate Functions
- Joins
- CASE statements
- Date functions

## Dataset Description
The dataset includes order-level data such as:
- Order Date
- State, City, Location
- Restaurant and Dish details
- Price, Rating, and Rating Count
